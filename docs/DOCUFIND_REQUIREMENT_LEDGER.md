# DocuFind Requirement Ledger

Read-only codebase assessment, 2026-07-21. Statuses reflect static inspection only; no build, unit test, emulator, device, or APK-install verification was performed.

## Architecture Understood

DocuFind is an Android app built with Jetpack Compose, Hilt, and MVVM-style screen ViewModels. It stores structured data in Room backed by SQLCipher (schema v12), schedules reminders through `AlarmManager`, and protects access with `AuthGate`. Attachments use `EncryptedFileManager` with AES-GCM and Android Keystore-wrapped keys. OCR uses ML Kit; QR scanning uses CameraX plus ML Kit; PDF export uses PdfBox.
